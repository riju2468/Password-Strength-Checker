# 🔐 Password Strength Checker

A modern and responsive Password Strength Checker and Password Generator built using **HTML, CSS, and JavaScript**.

## 🚀 Live Demo

👉 https://riju2468.github.io/Password-Strength-Checker/

---

## ✨ Features

- ✅ Check password strength instantly
- 🔒 Strong password generator
- 👤 Generate password from your name
- 👁️ Show/Hide password
- 📊 Visual strength meter
- ✔️ Password requirement checklist
- 📋 One-click copy to clipboard
- 📱 Responsive design for desktop and mobile

## 💡 How It Works

### Password Strength Checker

Evaluates passwords based on:

- Minimum length
- Uppercase letters
- Lowercase letters
- Numbers
- Special characters

### Password Generator

Generates secure random passwords of your preferred length.

### Name-based Password Generator

Creates a memorable password using your name mixed with numbers and special characters.
