Password Helper (static)

Files:
- index.html — main page
- style.css — styling
- script.js — password checks and generators

Open locally:

```bash
# From file explorer: double-click index.html
# or from terminal (Windows):
start "" "%USERPROFILE%\\c\\Python\\password_web\\index.html"
```

Notes:
- This is a static single-page app. All password checks and generation happen in the browser.
- No server required; you can host on GitHub Pages or any static host.
